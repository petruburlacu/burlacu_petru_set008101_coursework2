Install Angular CLI

  • npm install -g @angular/cli

Install Express

  • npm install express --save (in server directory)

Mongoose | Interact with MongoDB from Node.js

  • https://scotch.io/tutorials/using-mongoosejs-in-node-js-and-mongodb-applications
  • https://www.npmjs.com/package/mongoose
  • npm install mongoose --save

Bodyparser (Middleware before handlers)

  • https://github.com/expressjs/body-parser
  • npm install body-parser --save

Cors | Passing the data from Development server

  • https://www.npmjs.com/package/cors
  • npm install cors --save

Json WebToken npm / jwt tokens

  • https://github.com/auth0/angular-jwt
  • npm install jsonwebtoken (MEAN-Blog> root)

Nodemon

  • npm install -g nodemon
